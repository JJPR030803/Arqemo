# TODO Convention

One file per session, per phase:

```
TODO/Phase0/TODO-2026-03-13.md
TODO/Phase0/TODO-2026-03-15.md   ← revisiting Phase 0 later
TODO/Phase1/TODO-2026-03-14.md
```

**Latest date = current state.** Open it to remember where you left off.

Revisiting earlier phases is expected — drop a new dated file
when you return to correct or improve something.
